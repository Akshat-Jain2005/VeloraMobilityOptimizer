% ===== ABSTRACT =====
\begin{abstract}
This report presents a scalable optimization framework for the Velora Mobility Optimization problem, which involves assigning employee transportation requests to a heterogeneous fleet of vehicles while minimizing operational cost and service delays. The problem is modeled as a variant of the Heterogeneous Vehicle Routing Problem with Time Windows (HVRPTW), known to be NP-hard. 

To address computational intractability at realistic scales, we adopt a heuristic-based approach consisting of a greedy construction phase followed by a local-search based metaheuristic. Soft time window violations and employee priorities are incorporated using penalty-based cost modeling. The current implementation demonstrates feasibility, modularity, and extensibility, forming a strong baseline for further optimization and system integration.
\end{abstract}

% ===== SECTION 1: INTRODUCTION =====
\section{Introduction}

Efficient employee transportation is a critical operational requirement for large organizations, particularly in urban environments where traffic congestion, cost constraints, and service-level expectations coexist. Velora’s mobility problem focuses on optimally assigning employee pickup and drop-off requests to a heterogeneous fleet of vehicles while respecting operational constraints such as vehicle capacity, time windows, and service priority.

From an algorithmic perspective, this problem closely resembles the Heterogeneous Vehicle Routing Problem with Time Windows (HVRPTW), an extension of the classical Vehicle Routing Problem (VRP). Such problems are combinatorial in nature and are known to be NP-hard, making exact optimization methods computationally infeasible for realistic instance sizes.

This project focuses on designing a scalable, modular optimization pipeline that produces high-quality solutions within practical time limits. Rather than pursuing exact optimality, the emphasis is placed on heuristic construction and local-search based improvement strategies suitable for real-world deployment.
