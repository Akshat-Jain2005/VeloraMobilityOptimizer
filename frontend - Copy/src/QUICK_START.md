## 🎨 Customization

Want to change colors? Edit the CSS variables in `styles.css`:

```css
:root {
  --primary: #6366f1;      /* Change primary color */
  --secondary: #ec4899;    /* Change accent color */
  --success: #10b981;      /* Change success color */
  --warning: #f59e0b;      /* Change warning color */
  --danger: #ef4444;       /* Change error color */
}
```

## 🐛 Troubleshooting

### Styles not loading?
Make sure all CSS files are in the root directory:
- styles.css
- homePageStyles.css
- resultsSection.css

### Animations not smooth?
Check if hardware acceleration is enabled in your browser.

### Colors look different?
Make sure you're viewing in a modern browser (Chrome, Firefox, Safari, Edge).

## 💡 Tips

1. **Hover Effects**: Hover over cards, buttons, and badges to see animations
